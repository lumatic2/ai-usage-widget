AI Usage Widget — first run

This widget reads your local AI usage data.

On every refresh it reads:

  • ~/.codex/auth.json (Codex bearer token)
  • your claude.ai session cookie (after you sign in via the widget)

These tokens are used only to call the official Claude / Codex usage APIs. They are never written back to disk by this app, never sent anywhere else, and never leave your machine.

Which panels?

You can change this later from the gear icon (⚙) on the widget.
